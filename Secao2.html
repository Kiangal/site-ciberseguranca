Métodos de Invasão

Scanning

Usado para identificar portas abertas e vulnerabilidades em redes e sistemas.

Exemplo: Uso do Nmap para encontrar portas abertas.

SQL Injection

Inserção de comandos SQL em formulários para acessar bancos de dados.

Exemplo: Código SQL em campo de login.

XSS

Permite executar scripts maliciosos no navegador do usuário.

Exemplo: Script injetado via formulário.

Engenharia Social

Manipulação de pessoas para obter informações confidenciais.

Exemplo: E-mail falso se passando por contato.

Força Bruta

Tentativas repetidas de senha até encontrar a correta.

Exemplo: Testar várias combinações automaticamente.

Phishing

Engana vítimas para roubar dados ou instalar malware.

Exemplo: E-mail falso pedindo pagamento.

Ransomware

Bloqueia dados ou sistemas exigindo resgate.

Exemplo: Arquivos criptografados.

Ataques a APIs

Exploração de falhas em APIs para acesso indevido.

Exemplo: Uso de credenciais roubadas.

Vulnerabilidades CVE

Falhas conhecidas catalogadas publicamente.

Exemplo: Execução remota de código.

Exploits Automatizados

Códigos que exploram falhas automaticamente.

Exemplo: Metasploit em testes de segurança.
